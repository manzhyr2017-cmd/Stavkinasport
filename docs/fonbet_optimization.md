# Оптимизация под Fonbet и Рубли

## Изменения в системе (v2.1)

### 1. Перевод названий команд
Добавлен модуль `data/mappings.py` для автоматического перевода названий команд с русского (Fonbet) на английский (Dixon-Coles Model). Это позволяет использовать ML-модели на данных российского букмекера.

### 2. Валюта и Банкролл
- Валюта переключена на **RUB (₽)** во всех сообщениях и логах.
- Начальный банкролл установлен в **100,000 ₽**.
- Минимальная ставка снижена до **100 ₽**.
- Максимальная ставка на одно событие: **5% от банка**.

### 3. Настройки для Экспрессов
Сделан упор на формирование экспресс-ставок с высокими коэффициентами:
- **Разрешены высокие коэффициенты** (до 15.0) в экспрессах.
- **Снижен порог вероятности** до 15% для экспресс-событий, чтобы включить недооцененные события (Value Bets).
- **Приоритет надежности**: из всех доступных Value Bets выбираются самые вероятные.
- Алгоритм **сортирует события по вероятности**, собирая в экспрессы самые "безопасные" из валуйных исходов.
- Экспрессы до 5 событий с потенциально очень высоким коэффициентом.
- **AI Анализ**: К каждому экспрессу добавляется объяснение, почему выбраны именно эти исходы (например, "Высокая вероятность", "Сверх-валуй").

### 4. Работа с одним букмекером
- Отключена проверка через Pinnacle ("Sharp Bookmaker"), так как работаем только с Fonbet.
- Система теперь принимает сигналы, даже если они подтверждены только одним источником (Fonbet + ML модели).

## Запуск
Бот готов к работе.
- Запустить сканирование: `/scan`
- Посмотреть банкролл: `/bankroll`
- Получить экспрессы: `/express` (после сканирования)
